Build
Day 2-4
  • Draft AI agent prompt

Testing
Day 4-5
  • Internal test calls (20+ scenarios)
    Standard inquiries, edge cases, spam, wrong number, angry caller, after-hours, multi-question.
    Danny / Karim
  • Test all department transfers
    Verify each transfer route works. Confirm name verification before transferring.
    Danny / Karim
  • Test automation workflows
    Trigger each n8n workflow manually. Verify outputs, notifications, error branches.
    Claude
  • Client review call
    Walk client through agent capabilities. Let them test live. Gather feedback.

Go Live

  • Get client sign-off

    Danny / Karim
  • Move agent to production

    Claude + Danny/Karim
  • Send go-live confirmation to client
    Email: agent is live, here's what to expect, how to reach us for issues.
    Danny / Karim
  • Set up Stripe recurring billing
    Monthly subscription link sent to client. Auto-charge on the 1st.
    Danny / Karim
